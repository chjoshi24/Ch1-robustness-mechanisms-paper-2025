# Robustness mechanism manuscript code and data files
This repository contains R code and other related files for the robustness mechanism experiment. 

Preprint link: https://www.biorxiv.org/node/5131247.external-links.html 

## File information
* **Combined ant choices_low sucrose experiment.csv and Combined ant choices_high sucrose experiment.csv**: These csv files contain data for what choices ants made during each of the experiments (see the supplementary file S1 for more information on the columns)
* **Combined ant gaster angles_low sucrose experiment.csv and Combined ant gaster angles_high sucrose experiment.csv**: These csv files contain data on the average gaster angles for the ants that either made a correct choice or a wrong choice (see the supplementary file S1 for more information on the columns).
* **R code_robustness manuscript_2025.Rmd**: This is R markdown file that contains the entire annotated code. It uses the data from the four csv files described above.
* **Information on datasheet columns.pdf**: This PDF files contains information on individual columns in the above four datasheets.
